###Практическое задание 2

##Цель

Закрепить базовые навыки работы с файловой системой в Node.js: чтение файла,
обработка его содержимого и вывод результата в консоль.

- Реализованы синхронное и асинхронное выполнение команды чтения файла
- Видно, что асинхронные команды, выполняются позже синхронны, так как 
EventLoop помещает их в очередь на выполнение!
