##Практическое задание: знакомство с Vite

Цель:

Освоить базовую работу с Vite: создание проекта, запуск dev-сервера, организацию
структуры файлов, подключение стилей, работу со статическими ресурсами и сборку
проекта для production.

Задание (шаги):

Тема работы: создать небольшой одностраничный проект на Vite — например,
страницу-визитку, каталог карточек или мини-приложение “Список задач”.

##Для работы с режимами development и production использутся порт:3000
